# Run hardening: fixes for issues #40 and #42

One PR branch `fix/run-hardening-40-42` (from main 540c568) collecting the
engine and gate fixes discovered during the first fully supervised board run.
Sources: issue #40 (agent isolation, stall detection) and issue #42 (14
consolidated findings). Findings 13 and 14 are already fixed on main; finding
8 and the isolation half of #40 are already merged into this branch from the
pre-built fix branch (commits a35c81a, 11aef67).

## Orchestration

Coordinator does docs and delegation only. Each task runs in its own git
worktree on a branch `fix4042/<task>` cut from `fix/run-hardening-40-42`;
finished branches merge back into the PR branch sequentially. No pushes, no
PR creation without the owner's explicit approval.

clean. Commits: conventional, --signoff, Co-Authored-By trailer of the acting
model. The final merged branch runs the full workspace suite plus
